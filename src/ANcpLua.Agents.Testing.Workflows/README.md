# ANcpLua.Agents.Testing.Workflows

Consumer toolkit for Microsoft Agent Framework — bundling, governance, testing.

Stable workflow testing harness for Microsoft Agent Framework consumers, including workflow fixtures, event assertions, checkpoint helpers, and the framework-internals mirror.

Compatible with: Microsoft.Agents.AI 1.6.x
Tested against: Microsoft.Agents.AI 1.6.1
Capability tested against: Microsoft.Agents.AI.Workflows 1.4.0

Channel: stable. This package exposes a facade layer over Microsoft.Agents.AI.Workflows and may change internal implementations across versions, so consumers should track facade contract changes rather than diff internal internals for parity.
