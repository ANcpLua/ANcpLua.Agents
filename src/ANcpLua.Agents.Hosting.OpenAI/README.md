# ANcpLua.Agents.Hosting.OpenAI

Consumer toolkit for Microsoft Agent Framework — bundling, governance, testing.

Alpha-channel OpenAI-compatible hosting facades and client factory for Microsoft Agent Framework consumers.

Compatible with: Microsoft.Agents.AI 1.8.x
Tested against: Microsoft.Agents.AI 1.8.0
Capability tested against: Microsoft.Agents.AI.Hosting.OpenAI 1.8.0-alpha.260528.1 and Microsoft.Agents.AI.OpenAI 1.8.0

Channel: alpha. Keep this package isolated from stable and preview consumers unless explicitly intended.

> **Naming:** `Qyl*` = consumer-facing facade / entry-point, bare = primitive consumers may compose with. See [the convention in ANcpLua.Agents](../ANcpLua.Agents/README.md#naming-convention).
